### Тестовое задание: Интернет-магазин

#### Цель

Разработать простой интернет-магазин с возможностью просмотра товаров и оформления заказа.

---

## Основные требования

### Backend

Использовать один из фреймворков:

* FastAPI
* Django

### Frontend

Использовать серверный рендеринг:

* Django Templates
* Jinja2

Использование React/Vue/Angular не требуется.

---

## Функциональные требования

### Каталог товаров

Пользователь должен иметь возможность:

* Просматривать список товаров.
* Просматривать карточку отдельного товара.
* Видеть название, описание, цену и изображение товара.

### Оформление заказа

Пользователь должен иметь возможность:

* Добавлять товары в корзину.
* Просматривать содержимое корзины.
* Оформлять заказ.

При оформлении заказа необходимо указать:

* Имя клиента.
* Номер телефона.

После оформления заказ должен сохраняться в базе данных.

---

## База данных

Использовать любую реляционную СУБД:

* PostgreSQL (предпочтительно)
* SQLite

Минимальные сущности:

### Product

* id
* name
* description
* price
* image

### Order

* id
* customer_name
* phone_number
* created_at

### OrderItem

* id
* order_id
* product_id
* quantity

---

## Контейнеризация

Проект должен запускаться через Docker Compose.

Команда запуска:

```bash
docker compose up --build
```

После запуска приложение должно быть доступно локально без дополнительных настроек.

# Дополнительные задания (бонус)

### 1. Авторизация по электронной почте

Реализовать регистрацию пользователя с подтверждением email.

Будет плюсом:

* Отправка писем через SMTP.
* Использование фоновых задач для отправки писем.

### 2. Административная панель

Реализовать возможность:

* Создавать товары.
* Редактировать товары.
* Удалять товары.
* Просматривать список заказов.

### 3. Деплой

Развернуть проект на любом публичном хостинге:

* Render
* Railway
* Fly.io
* VPS
* Другой вариант

В README указать ссылку на работающий проект.

### 4. Тестирование

Написать автоматические тесты для основных сценариев:

* Получение списка товаров.
* Создание заказа.
* Авторизация пользователя.
